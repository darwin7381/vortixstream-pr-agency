## 🎯 V1 與 V2 的差異

| 功能 | V1 (當前) | V2 (規劃中) |
|------|-----------|-------------|
| 資料來源 | 前端假資料 | 後端 API + 資料庫 |
| 下載功能 | 假提示訊息 | 真實檔案下載 |
| 表單提交 | Console log | 儲存到資料庫 |
| AI 功能 | 預告展示 | 實際可用 |
| 編輯器 | 無 | 線上編輯器 |
| 專案管理 | 無 | 我的專案列表 |
| 使用額度 | 無 | AI 次數限制 |

## 🔄 後續整合步驟（V2）

### 後端開發
1. 創建資料庫 Schema
   - `pr_templates` 表
   - `template_downloads` 表

2. 實作 API 端點
   - `GET /api/templates`
   - `GET /api/templates/{id}`
   - `POST /api/templates/{id}/download`
